 🐍 Python Fundamentals - Algorithms Collection

Este repositorio contiene una colección completa de algoritmos desarrollados en Python, enfocados en el aprendizaje de lógica de programación.

##  Contenido

###  Básicos

* Operaciones aritméticas
* Comparaciones
* Validaciones

###  Condicionales

* Clasificación de datos
* Sistemas de decisión (IMC, ISR, boletos)

###  Ciclos

* Números pares e impares
* Sumatorias
* Tablas de multiplicar

###  Acumuladores

* Promedio de alumnos
* Promedio de grupos
* Análisis de datos

###  Simulación de problemas reales

* Censos poblacionales
* Sistemas de transporte
* Descuentos comerciales
* Cálculo de impuestos

##  Tecnologías

* Python 3

##  Objetivo

Desarrollar habilidades en:

* Lógica de programación
* Estructuras de control
* Resolución de problemas
